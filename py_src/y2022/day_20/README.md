rotating lists

positive

```python
l = [1, 2, -3, 3, -2, 0, 4]
l[:1], l[1:] = l[-1:], l[:-1]
l # [4, 1, 2, -3, 3, -2, 0]
```

negative

```python
l = [1, 2, -3, 3, -2, 0, 4]
l[-1:], l[:-1] = l[:1], l[1:]
l # [2, -3, 3, -2, 0, 4, 1]
```
